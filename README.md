# 🛒 PriceWise

**PriceWise** is a smart web application designed to help users in Sri Lanka make better shopping decisions by comparing product prices across multiple supermarkets.

🚀 Built with **React**, PriceWise combines price comparison, recipes, AI assistance, and store location features into one powerful platform.

## ✨ Key Features

### 🔍 Price Comparison
Compare product prices across:
- Cargills
- Keells
- Glomark  

Users can choose products based on **price** and **reviews**.

---

### 🛒 Flexible Shopping
- Buy products **online**
- Or select **store pickup** based on convenience

---

### 🍲 Sri Lankan Recipes
- Browse authentic Sri Lankan recipes  
- Automatically get the **best-priced ingredient packages** by comparing supermarket prices

---

### 📰 Price News & Updates
- Stay updated with **price changes**
- Discover **new and existing products**

---

### 🤖 AI Chatbot
- Helps users find the **best deals**
- Provides smart recommendations

---

### 📍 Store Locator
- Find nearby supermarket branches  
- Access **contact details** across Sri Lanka

---
